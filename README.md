<h1 align="center">:page_with_curl: Descrição do Projeto</h1>

<p>Objetivo do projeto e a criação de um catálogo de quadrinhos junto de um buscador de personagens ou quadrinhos sendo focado no treinamento de Reactj,Redux e Style-Components
  tendo integração com api da <a href="https://developer.marvel.com/">Marvel Api</a> para os dados serem mostrados na tela.</p>


<h1 align="center">:notebook: Funcionalidades</h1>

- [x] Retorno de todos os Quadrinhos e Personagens
- [x] Paginação atraves de um botão para carregar mais conteudo
- [x] Quadrinho ou Personagem detalhes mostrando alguns detalhes dos mesmo 
- [x] Procurador de Quadrinhos ou Personagem com base no nome  
- [x] Pagina de Personagem
- [x] Buscador na pagina de Personagens para achar aquele personagem com nome desejado 


<h1>:pushpin: Como Usar</h1>

<p>Para usar esse projeto basta criar uma conta na Marvel Api e pegar os seus Ts,apikey,hash e substituir no arquivo service.jsx que esta na pasta service</p>

<p>Por Exemplo: params: {
    ts: "Seu ts aqui",
    apikey: 'Sua api key aqui',
    hash: 'Seu Hash aqui',
  },</p>

<p>Feito isso basta apenas rodar npm start ou yarn start</p>
